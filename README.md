# Instagram Multi-Account Viewer

複数のInstagramアカウントを一画面で管理・閲覧するためのシンプルなWebダッシュボードです。

## 機能

- 最大5つのInstagramアカウントをカード形式で一覧表示
- - 各アカウントのプロフィール、フィード、ストーリーへのクイックアクセス
  - - 全アカウントを一括でタブに開く機能
    - - レイアウト切り替え（自動 / 1列 / 2列 / 3列）
      - - アカウント設定はブラウザのローカルストレージに保存（外部送信なし）
        - - レスポンシブ対応
         
          - ## 使い方
         
          - ### GitHub Pagesで利用する場合
         
          - 1. このリポジトリのSettings > Pages から GitHub Pagesを有効にする
            2. 2. Source を `main` ブランチに設定
               3. 3. 公開されたURLにアクセス
                  4. 4. アカウントのユーザー名を入力して「ダッシュボードを表示」をクリック
                    
                     5. ### ローカルで利用する場合
                    
                     6. 1. `index.html` をダウンロード
                        2. 2. ブラウザで `index.html` を直接開く
                           3. 3. アカウントのユーザー名を入力して使用開始
                             
                              4. ## 注意事項
                             
                              5. - Instagramはiframeでの埋め込みをブロックするため、各アカウントは「新しいタブで開く」ボタンからアクセスします
                                 - - データはすべてブラウザのローカルストレージに保存され、サーバーへの送信は一切行いません
                                   - - このツールはInstagramの非公式ツールです
                                    
                                     - ## 技術構成
                                    
                                     - - HTML / CSS / JavaScript のみ（フレームワーク不使用）
                                       - - 外部依存なし
                                         - - GitHub Pages対応
